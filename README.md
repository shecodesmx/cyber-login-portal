# 🔐 Cyber Login Portal

Interfaz futurista de **login y registro** con estilo cyberpunk, creada con HTML, CSS y JavaScript.

---

## 🚀 Tecnologías

- HTML5
- CSS3
- JavaScript

---

## ✨ Características

- 🔐 Formulario de login y registro
- 🌌 Diseño cyberpunk/futurista
- ⚡ Animaciones suaves
- 🎨 UI moderna
- 📱 Responsive design

---

## 📂 Estructura del proyecto

```
cyber-login-portal/
│
├── Cyber_Login_Portal.html
├── Register.html
├── style.css
├── script.js
```

---

## ▶️ Cómo usar

1. Descarga el proyecto
2. Abre `Register.html` en tu navegador
3. Abre `Cyber_Login_Portal.html` en tu navegador
